# Inhalt der Anwendung

- Es handelt sich um eine webbasierte Client-Server Anwendung, die das Ziel hat, Notizen auf eine sichere Art und Weise zu erstellen
- Man kann sich registrieren und Notizen erstellen, die entweder öffentlich oder privat sind
- Die Notizen unterstützen die Verwendung von Markdown und HTML
- Den Notizen kann man außerdem auch ein YouTube-Video anhängen
- Dabei wird der Datenschutz mit Hilfe der Zwei-Klick-Methode sichergestellt
- Öffentliche Notizen können über eine Suchfunktion gesucht werden
- Private Notizen können nur von dem Ersteller der Notizen betrachtet werden
- Wenn man das Passwort vergessen haben sollte, kann man mit Angabe der E-Mail sein Passwort ändern
